FizzBuzz!

by Bruce Cox

This is a version of a FizzBuzz game that counts from 1 to 100. For every number divisible by 3, it prints "Fizz", and for every number divisible by 5, it prints "Buzz". If a number is divisible by both 3 AND 5, it will print "FizzBuzz". If the number meets none of this criteria, it will simply print the number. Enter a number between 1 and 100, then click "Print FizzBuzz" to see the results.
